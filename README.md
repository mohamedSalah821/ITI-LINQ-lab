# ITI LINQ Lab Collection

This repository contains solutions for LINQ labs completed during the ITI training program.  
It includes multiple parts of exercises exploring LINQ queries in C# with **Method Syntax** and **Query Expression Syntax**.

## Project Overview

- **Part 01** – Basic LINQ queries on simple collections  
- **Part 02** – LINQ queries with lists, arrays, and more complex data structures  
- **Part 03** – Advanced queries involving books, authors, publishers, and subjects  
- **Part 04** – Additional exercises and creative LINQ challenges  

## Features

- Querying and filtering data using LINQ  
- Grouping, ordering, and joining collections  
- Working with nested objects (Books, Subjects, Publishers, Authors)  
- Demonstrating both **Method Syntax** and **Query Expression Syntax**
